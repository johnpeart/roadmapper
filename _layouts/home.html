Roadmapper

Use this tool to visualise a high-level roadmap based on data from a tab-separated value file.

Create a roadmap

Before you start

To use this service, you will need a tab-separated value (.tsv) file containing information about your roadmap. A template is available to get you started.

Download a TSV template

How is the data processed?

All data is processed on your device, in your browser. The data never leaves your device.
